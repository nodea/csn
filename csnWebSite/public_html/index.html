Presentation

This wiki works as a information repository and documentation of our research about Client-Side Adaptation. Our work aims to allow users to adapt (augment) existing Web Applications with changes in contents, functionalities, etc. in order to support users in their navigation among the Web. In this context, we have developed as a proof of concept, our Framework for Client-Side/Concern-Sensitive Adaptation which can be extended with different kinds of artifacts. Although the Framework provide some base componenets, the main idea is that these artifacts must be developed by the user community. Then, the framework foresees two kinds of users:


  • Final users with programming skills: for developing new Framework's extensions.
  • Final users: who can profit from both the Framework and these extensions for supporting their tasks.


How I have to read this wiki?


This wiki is structured in different parts:


The first part is a theorical background for those interested reader.

Then, we provide documentation for developers (users with programming skills)

We show a kind of manual of use for final users.

You may download both the Framework and augmenters from the download page.
